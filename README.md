# Econ 320 Homework 3 
By: Sydney Gitlitz

For this assignment, I used Medicare Advantage data from the class Github. This assignment analyzes Medicare Advantage plan data from 2010–2015, focusing on how star ratings relate to market outcomes. The project includes summary statistics, regression analysis, and a regression discontinuity (RD) design to estimate the causal effect of star ratings on enrollments.
-Functions.py was used to define the function used in the code
